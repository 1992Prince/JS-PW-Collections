âœ… List of Common Playwright Test APIs:

| Keyword / Function  | Purpose & Use Case                                                           |
| ------------------- | ---------------------------------------------------------------------------- |
| `test`              | Define a test case.                                                          |
| `test.describe()`   | Group multiple related tests.                                                |
| `test.beforeAll()`  | Run **once** before all tests in a group or file.                            |
| `test.afterAll()`   | Run **once** after all tests in a group or file.                             |
| `test.beforeEach()` | Run **before every test** (for common setup like login, navigation, etc).    |
| `test.afterEach()`  | Run **after every test** (for cleanup like logout, clear storage, etc).      |
| `test.skip()`       | Skip a test conditionally or unconditionally.                                |
| `test.only()`       | Run **only** this test (used for debugging).                                 |
| `test.fixme()`      | Mark a test that is broken or needs work.                                    |
| `test.fail()`       | Mark test as expected to fail (e.g., known bug).                             |
| `test.slow()`       | Tells Playwright to **increase timeout** for this test.                      |
| `test.step()`       | Used to **log logical steps** inside a test in trace viewer or report.       |
| `expect()`          | Assertion API (comes from `@playwright/test`)                                |
| `test.use()`        | Temporarily override fixtures or context options per test or describe group. |
