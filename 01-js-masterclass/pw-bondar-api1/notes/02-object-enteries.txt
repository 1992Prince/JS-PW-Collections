ðŸ”¹ Step 2: Object.entries(this.queryParams)

Object.entries(obj) â†’ ek object ko array of [key, value] pairs mein convert karta hai.

Example:

// Output: [ ['userId', 101], ['limit', 10] ]


ðŸ”¹ Step 3: for (const [key, value] of Object.entries(...))

Upar wale example mein:

Pehla iteration â†’ key = 'userId', value = 101

Dusra iteration â†’ key = 'limit', value = 10


ðŸ”¹ Step 4: url.searchParams.append(key, value.toString())

url ek JavaScript ka URL object hota hai.

Example: const url = new URL("https://api.example.com/data");

url.searchParams.append(k, v) â†’ query string mein naya param add karta hai.
Example:

url.searchParams.append("userId", "101");
url.searchParams.append("limit", "10");


Ab url ka value ban jayega:

value.toString() isliye use kiya gaya hai kyunki query string ke params hamesha string type hote hain.
