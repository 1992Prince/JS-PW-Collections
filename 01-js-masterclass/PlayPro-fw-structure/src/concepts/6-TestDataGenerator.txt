2. Random Integer
        static generateRandomInteger(min: number, max: number): number {
          return Math.floor(Math.random() * (max - min + 1)) + min;
        }
        
        
        ðŸ‘‰ Ye method min aur max ke beech ek random integer deta hai.
        Example:
        
        generateRandomInteger(1, 10) â†’ koi bhi number 1 se 10 ke beech.
        
        Formula explanation:
        
            (max - min + 1) â†’ range calculate hoti hai.
            Math.random() se 0â€“1 ka number milta hai.
            Multiply + add karne se range shift ho jaati hai.
